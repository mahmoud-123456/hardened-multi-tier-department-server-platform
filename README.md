* Hardened Multi-Tier Department Server Platform

## Project Overview
This project demonstrates a secure multi-tier server architecture including Infrastructure, Application, Database, and Automation.

## Architecture
The system is divided into three main tiers:
- Web Tier (Application Server)
- Database Tier (Internal Server)
- Security & Automation Layer

## Project Structure
- infrastructure: VM setup, networking, firewall
- application: web server and deployment
- database: database setup and management
- automation: security and backup scripts

## Team Roles
- Infrastructure & Network Engineer
- Application & Database Administrator
- Automation & Security Specialist

## Technologies Used
- Linux (Ubuntu Server)
- Nginx / Apache
- MySQL / PostgreSQL
- Bash scripting
- GitHub

## My Role
As the Application & Database Administrator, I was responsible for:
- Database setup and configuration
- Linking the application with the database
- Managing users and permissions
- Implementing backup and recovery solutions

## Pro Tips for Success
- Use GitHub/GitLab to store scripts and configurations
- Document all open ports with justification
- Perform failure simulation and recovery testing
